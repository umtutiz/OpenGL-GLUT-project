# 🎯 2D Target Shooting Game (OpenGL)

## 📖 About The Project
This repository contains a 2D target shooting game developed using **OpenGL** and the **GLUT** framework. Originally created as an academic project, it demonstrates the fundamentals of computer graphics, rendering pipelines, and user input handling.

## ✨ Key Features
* **2D Graphics & Rendering:** Utilizes OpenGL primitives to draw and render the game environment and targets.
* **Input Handling:** Processes user keyboard and mouse interactions for aiming and shooting mechanics via GLUT callbacks.
* **Core Game Loop:** Implements basic game state management, including target spawning and hit detection.

## 🛠️ Built With
* **Graphics API:** OpenGL
* **Framework:** GLUT (OpenGL Utility Toolkit)
* **Language:** C / C++
